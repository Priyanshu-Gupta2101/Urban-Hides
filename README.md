# Urbanhides

A leatherware shopping website.
This is the web application, containing the NextJS frontend and the Express API.

## Setup:

- #### API

- cd into /api

```
npm i
```

- Run the server

```
npm run server
```

- #### NextJS

- cd into `client`

- Install dependency modules

```
npm i
```

- Start server

```
npm run dev
```
